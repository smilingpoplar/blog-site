凯利公式

设赢时净收益率Rw，输时净收益率Rl（Rl取绝对值，确保正数）

例如：赢的话1变3，输的话1变0，输赢概率各50%，上述就是 $(20.5-10.5)/(2*1) = 0.25$

另：赌博中术语，赔率 = Rw / Rl

凯利公式的推导

假设本金为W0，n次抽样的增长倍数分别为M1,M2,M3 … Mn，则总收益

$$
W_n = W_0M_1M_2…M_n = e^{lnW_0+lnM_1+lnM_2+…+lnM_n} \ = e^{lnW_0 + nE(lnM)+ o(n)}（大数定律）= e^{lnW_0} * e^{nE(lnM) + o(n)} ≈ W_0 * e^{n*E(lnM)}
$$

总收益最大化就是E(lnM)最大化。不妨设投资比例为f，净盈利百分比a概率p，净亏损百分比b概率q=1-p，则

$E(lnM) = pln(1+af) + qln(1-bf)$，最大值当导数为0时取得，$\frac{ap}{1+af}-\frac{bq}{1-bf}=0$，得 $f = \frac{ap-bq}{ab}$

参考：Kelly Criterion in detail

设投资比例为f，n次抽样的净收益率分别为ri，净收益率均值为μ方差为σ^2，使用泰勒公式$ln(1+x)=x - x^2/2 + …$展开两次，
$$E(lnM) = E[ln(1+fr_i)] = E[fr_i - \frac{1}{2}f^2r_i^2 + …] ≈ E(fr_i) - E(\frac{1}{2}f^2r_i^2) = fμ - \frac{1}{2}f^2σ^2 $$
最大值当导数为0时取得，$μ - fσ^2 = 0$，得 $f = \frac{μ}{σ^2}$

凯利公式的应用

假设市场平均市盈率水平长期而言是稳定的，即涨和跌的概率都是50%，涨幅a和跌幅b对称 (1+a)(1-b)=1，由凯利公式，x=(0.5a-0.5b)/ab=0.5，采用半仓动态再平衡。

参考：基于市场平均市盈率与凯利公式的仓位控制策略

通过止盈止损线将股票投资转换为连续赌博：某种信号发出算入场点，设置止盈止损线，碰到止盈止损线算出场点，用历史回测计算止盈出场和止损出场的次数，这样就知道了赢时净收益率（止盈线）、输时净收益率（止损线）、赢的概率（止盈出场数/总出场数）、输的概率（1-赢的概率）
